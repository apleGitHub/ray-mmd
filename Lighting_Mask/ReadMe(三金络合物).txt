【使用方法】
1.请先备份你手上的ray，避免操作失误导致数据丢失。
2.将压缩包内的内容直接移动覆盖到你的ray根目录下。
（完成后【Light_Mask】文件夹应与原【Light】文件夹在同一目录下）
3.载入【Light_Mask】内的灯光时可以看到多出【SJ_MaskMap】栏目，
  为你想要正常照亮的材质赋予【SJ_Mask_On.fx】
  为你不想照亮的材质赋予【SJ_Mask_Off.fx】

TIPs 必要时可以打开【SJ_Mask_On.fx】或【SJ_Mask_Off.fx】自行修改参数，见下方【参数定义说明】
其余常规灯光操作可照常执行。

带遮罩栏目的ray灯光

ver1.0(152)
##################

【灯光说明】
本灯光基于ray1.5.2原版灯光增加遮罩栏目，可以控制灯光对单个材质的影响。
同时表情栏【Hide】可以隐藏灯光本体。

###基于ray1.5.2，其他版本会出错，请等待适配###

【使用规约】
1.请在借物表标注我的名字（三金络合物）
2.【禁止】不尊重作者的行为如占为己有等
3.【允许】二次配布，但是配布时必须包含这份规约和完整原始材质的压缩包，且借物表必须书写完整
4.【禁止】售卖！

【参数定义说明】

//透明阈值
float alphaThr = [float value 0.0 - 1.0];
根据贴图与材质透过率移除不需要的像素，与main文件种对于参数同理。


//遮罩强度
float maskThr = [float value 0.0 - 1.0];
遮罩的强度，0.0为完全不受光照影响，1.0为完全受光照影响
